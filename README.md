## 🚀 Local Installation & Quickstart

Deploy the entire telemetry framework onto your local hard drive completely offline by executing the following commands in your computer terminal:

### 1. Initialize the Workspace Folder
```bash
git clone https://github.com
cd wyvern-pollinator
```

### 2. Install Project Dependencies
Ensure you have **Node.js 24+** installed natively on your machine, then compile your project configuration lockfiles:
```bash
npm install
```

### 3. Execute the Telemetry Grid Simulation Scan
```bash
node engine.js
```

Upon a successful completion pass, the engine will automatically compile an interactive web dashboard folder on your computer disk named **`wyvern_nft_gallery.html`**, displaying all uniquely serialized, event-responsive vector NFT assets inside your standard desktop browser window.

## 🧪 Automated Continuous Integration

This repository includes a native **GitHub Actions Workflow** file located inside `.github/workflows/`. Every time you push a code adjustment or submit a pull request, an isolated cloud machine container will automatically initialize, pull your package locks, and execute a live evaluation sweep to verify system-level health.

> [!NOTE]
> Ensure that `package-lock.json` is always committed right alongside your `package.json` file in the root directory. If the lockfile is absent, the automated `npm ci` cloud runner will instantly crash to prevent unauthorized package drift.
